# Galaxy CV — 1Y Frontend

A minimal, space-themed personal portfolio built with React + Vite. Designed to showcase frontend fundamentals with clean, responsive UI and subtle animations.

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Features

- **Component-based** — Modular structure under `src/components/`
- **Dark space theme** — Animated starfield, gradient accents, deep color palette
- **Responsive** — Mobile-first layout that scales from 320px to 1920px+
- **Smooth interactions** — Hover effects, scroll behavior, form feedback
- **Minimal dependencies** — React + Vite only (no heavy libraries)

## Build for Production


npm run build
npm run preview
